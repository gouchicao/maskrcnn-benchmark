# [maskrcnn-benchmark](https://github.com/facebookresearch/maskrcnn-benchmark)

## 环境
* Ubuntu 18.04
* CUDA 10
* CUDNN 7
* Python 3.6.8

## 镜像
1. 拉取镜像
```bash
$ sudo docker pull gouchicao/maskrcnn-benchmark:latest
```

2. 构建镜像
```bash
$ sudo docker build -t gouchicao/maskrcnn-benchmark:latest docker/
```
